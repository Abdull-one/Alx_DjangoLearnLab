
# Admin Interface Configuration

## Registering the Book Model
Registered the Book model with the Django admin interface in `bookshelf/admin.py`.

## Customizing the Admin Interface
Customized the admin interface to display title, author, and publication year in the admin list view.
Configured list filters and search capabilities to enhance the admin’s usability for Book entries.

## Admin Interface Features
* List display: title, author, publication year
* List filter: author, publication year
* Search fields: title, author
